ชื่อ-สกุล: ธิติวุฒิ ศรีอมรรัตน์

สถานที่แนะนำ แนะนำสถานที่ใหม่ เกี่ยวกับเว็บไซต์

สถานที่แนะนำในมหาวิทยาลัยขอนแก่น

1. สำนักหอสมุด มหาวิทยาลัยขอนแก่น

อ่านหนังสือ ทำงานกลุ่ม

ศูนย์กลางการเรียนรู้ที่รวบรวมหนังสือ วิทยานิพนธ์ วารสาร และฐานข้อมูลออนไลน์ รองรับนิสิตนักศึกษาทุกคณะ มีทั้งโซนอ่านเงียบและโซนอ่านแบบทำงานกลุ่ม พร้อมปลั๊กไฟและอินเทอร์เน็ตความเร็วสูง

2. ศูนย์อาหารคอมเพล็กซ์ (Complex)

อาหาร นั่งชิล

แหล่งรวมอาหารยอดฮิตของนักศึกษา มีทั้งอาหารตามสั่ง ก๋วยเตี๋ยว เครื่องดื่ม และของหวาน ราคานักศึกษา เดินทางสะดวก อยู่ใกล้หลายคณะ เหมาะทั้งสำหรับกินข้าวและนัดเพื่อนคุยงาน

3. บึงสีฐาน

พักผ่อน ออกกำลังกาย

จุดพักผ่อนยอดนิยม บรรยากาศริมบึง เหมาะสำหรับเดินเล่น วิ่งออกกำลังกาย หรือถ่ายรูป ตอนเย็นจะสวยเป็นพิเศษเพราะเห็นพระอาทิตย์ตกดิน

4. ประตูศรีฐาน

แลนด์มาร์ก ถ่ายรูป

แลนด์มาร์กสำคัญของมหาวิทยาลัยขอนแก่น เป็นอีกหนึ่งจุดที่นิสิตนิยมมาถ่ายรูป โดยเฉพาะช่วงรับน้องและรับปริญญา

แบบฟอร์มแนะนำสถานที่ใน มข.

หากคุณมีสถานที่ที่อยากแนะนำให้เพื่อน ๆ รู้จัก สามารถกรอกแบบฟอร์มด้านล่างนี้ได้ ข้อมูลจะถูกส่งไปยังผู้ดูแลเว็บไซต์ผ่านบริการ Web3Forms
