# KKU Campus Guide

Welcome to the KKU Campus Guide project! This project is designed to provide an informative and visually appealing guide to various locations within Khon Kaen University (KKU). 

## Project Structure

The project consists of the following key components:

- **docs/lab02_guide/index.html**: The main HTML document that introduces at least four locations within KKU. Each location includes descriptions and images, along with the student's ID and full name.

- **docs/lab02_guide/css/style.css**: The CSS file that styles the webpage. It utilizes the Box Model for layout and includes media queries to ensure the website is responsive across different devices.

- **docs/lab02_guide/images/**: This directory contains images for each location:
  - **location1-1.jpg** and **location1-2.jpg**: Images for the first location.
  - **location2-1.jpg** and **location2-2.jpg**: Images for the second location.
  - **location3-1.jpg** and **location3-2.jpg**: Images for the third location.
  - **location4-1.jpg** and **location4-2.jpg**: Images for the fourth location.

## Viewing the Website

To view the KKU Campus Guide, open the `index.html` file in a web browser. You can navigate through the various locations and learn more about each one.
